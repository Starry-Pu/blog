## 👋 个人简介 (Intro)

**你好！我是 Pu Starry**

我是一名前端开发工程师，拥有 **5 年**的开发经验。

虽然我的大学专业是**机械制造及其自动化** ⚙️，但出于对编程的热爱，我自学并转型成为了一名专职的前端开发者。

在工作中，我曾通过优化渲染逻辑将页面响应速度提升 **30%**，也习惯于深入源码（如 Element UI 和 Vue 生命周期）去解决那些棘手的数据回显和交互难题。

在工作之外，我是一个热爱生活、喜欢“折腾”的开源爱好者。

## 🛠️ 技术栈与折腾 (Tech & Tinkering)

我的技术雷达主要覆盖：

* **核心栈**：Vue.js (全家桶) / JavaScript / CSS
* **进阶探索**：熟悉 React 及其 Hooks 生态，同时也涉猎 Node.js 和服务端渲染。
* **工程化**：习惯于封装通用的 Axios 拦截器和组件库，致力于减少冗余代码，提升项目的可维护性。

我不仅仅满足于“能用”，更在意“好用”。无论是将首页加载时间压缩到 2 秒以内，还是解决多端设备的 97% 兼容性问题，我都乐在其中。
